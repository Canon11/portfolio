# Portfolio

## Description

My portfolio developed by me using Vue.js and Vuetify.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Deploy

```
npm run deploy
```

### Lints and fixes files

```
npm run lint
```
